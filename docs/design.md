\# Diseño visual de AndyAzhTEC Classroom



\## Objetivo visual



AndyAzhTEC Classroom debe mantener una identidad visual coherente con la página principal de AndyAzhTEC.



La plataforma debe verse moderna, tecnológica, clara y profesional, pero sin perder la estética neon/gaming que ya identifica a la marca.



\---



\# Identidad base



La estética principal será:



\- tech

\- gaming

\- neon

\- educativa

\- moderna

\- clara de usar

\- visualmente potente



El sistema debe sentirse como una plataforma propia de AndyAzhTEC, no como una plantilla genérica.



\---



\# Referencia visual



La referencia principal será la página actual de AndyAzhTEC.



Elementos a sostener:



\- paleta cyan / violeta / lila

\- fondos oscuros con profundidad

\- tarjetas con transparencias

\- bordes suaves

\- efectos glow moderados

\- tipografía futurista para títulos

\- tipografía legible para contenido

\- iconografía tecnológica

\- estilo dashboard moderno



\---



\# Tipografías
