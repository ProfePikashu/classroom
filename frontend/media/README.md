# Media de AndyAzhTEC Classroom

Carpeta para recursos propios del proyecto.

## Estructura

- images: imágenes propias del Classroom.
- icons: íconos personalizados.
- videos: videos propios o clips locales.
- sounds: sonidos de interfaz.
- gifs: GIFs propios.

## Sonidos sugeridos

Para que funcionen los ruiditos de click, cargar archivos como:

- frontend/media/sounds/ui-click.mp3
- frontend/media/sounds/ui-open.mp3
- frontend/media/sounds/ui-success.mp3

Si los archivos no existen, el sistema no rompe.
